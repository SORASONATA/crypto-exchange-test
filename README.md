# Crypto Exchange Backend

This project is a backend system for a simple cryptocurrency exchange platform.
The system allows users to create accounts, trade cryptocurrencies using fiat
currency (THB, USD), and transfer assets between users.

This project was developed as a backend assignment focusing on:
- Database design (ER Diagram)
- RESTful API design
- Model relationships
- Controller and routing structure
- Seed data for testing

How to run this project
step 1 : Clone repository
```bash
git clone https://github.com/SORASONATA/crypto-exchange-test.git
cd crypto-exchange
````
step 2 : Install dependencies
```bash
npm install
```
step3 : Run database seed
```bash
node seeders/seed.js
```
step 4 : Start the server
```bash
node app.js
```
step 5 : test
```bash
http://localhost:3000/orders
http://localhost:3000/users
```
